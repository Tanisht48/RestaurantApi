# RestaurantApi
___
## FrameWork and Language
    Framework:SpringBoot
    Language : Java
## Dependencies
    SpringBoot Starter Web
    Lombok
    Validation
## Data Flow
  ### Controller
       1. @PostMapping("restaurant")
       2. @PostMapping("restaurants")
       3. @GetMapping("restaurants")
       4. @GetMapping("restaurant/{id}")
       5. @PutMapping("restaurant/{iD}/speciality")
       6. @DeleteMapping("restaurant/{iD}")
  ### Service
       1. Get all restaurant's
       2. Get a specific restaurant
       3. add restaurant
       4. add multiple restaurant
       5. update a specific Restaurants speciality
       6. delete restaurant
## Data Structure
  ArrayList from the collection framework

## Project Summary
  In this application we can  Create Read Update Delete a Restaurant and its Properties from the different api that are in this application
